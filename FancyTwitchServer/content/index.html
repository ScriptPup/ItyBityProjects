Fancy Twitch Services
Setup This link will bring you to the setup page, which is where you can define your twitch application details, and sign-in as the bot account you want to use.
http://localhost:9000
Chat Relay Use this link to go to a web-page which will simply spit all chats out.

Use the paramaters (or fill in the boxes below and use the link under the section title): ?channel=[mychannel]&fade=[fade_timeout]&swipe=[true|false] to control behavior
  • Channel = The Twitch channel you wish to monitor
  • Fade Timeout = The time (in seconds) to keep messages before fading them away
  • Swipe = When enabled fading messages will be removed from the screen with a subtle "swipe" effect
http://localhost:9000
Showcase Use this link to go to a web-page which will display whatever art work was last (default) redeemed

Use the paramaters (or fill in the boxe below and use the link under the section title): ?pos=[number] to determine which redemption to gather (starts at 0 and goes UP to get redemptions further in the past. 0 if most recent, 1 is one before most recent, etc.)
Showcase Setup This link will bring you to the showcase setup page. Here you can view past redemptions, and manually redeem something yourself (without using channel points )
