Fancy Twitch Services
Setup This link will bring you to the setup page, which is where you can define your twitch application details, and sign-in as the bot account you want to use.
Chat Relay Use this link to go to a web-page which will simply spit all chats out.

Use the paramaters: ?channel=[mychannel]&fade=[fade_timeout]&swipe=[true|false] to control behavior

In the future, may add some helper pop-up before redirecting to make this more convenient (like kapchat does).
Showcase Use this link to go to a web-page which will display whatever art work was last (default) redeemed

Use the paramaters: ?pos=[number] to determine which redemption to gather (starts at 0 and goes UP to get redemptions further in the past. 0 if most recent, 1 is one before most recent, etc.)

In the future, may add some helper pop-up before redirecting to make this more convenient.
Showcase Setup This link will bring you to the showcase setup page. Here you can view past redemptions, and manually redeem something yourself (without using channel points )

Use the paramaters: ?pos=[number] to determine which redemption to gather (starts at 0 and goes UP to get redemptions further in the past. 0 if most recent, 1 is one before most recent, etc.)

In the future, may add some helper pop-up before redirecting to make this more convenient.
